If you provided observations, you may want to check the documentation of the MPC astrometry format. The format is a little complicated, mostly because all data must be forced into 80 columns so that it can fit on a punched card. Make sure that all columns are properly aligned.

If you're only going to use the observations in Find_Orb, you may find some of the Find_Orb extensions to the astrometric format to be useful. In particular, this can help if your times are in hour/minute/second form and you don't want to convert to decimal days, or if your positions aren't in base-60 (Babylonian/sexagesimal) form.
